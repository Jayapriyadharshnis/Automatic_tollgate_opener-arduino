Automatic Toll Gate Opener - Components List

1. Arduino Uno
2. Ultrasonic Sensor (HC-SR04)
3. Servo Motor (SG90)
4. 16x2 LCD Display with I2C module
5. Breadboard
6. Jumper wires (Male-to-Male, Male-to-Female)
7. USB cable for Arduino power and programming
8. External power source (optional)

Pin Connections:

Ultrasonic Sensor:
Trigger → Pin 9
Echo → Pin 8
VCC → 5V
GND → GND

Servo Motor:
Signal → Pin 3
VCC → 5V
GND → GND

LCD with I2C:
SDA → A4
SCL → A5
VCC → 5V
GND → GND
